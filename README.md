## 📌 Learning Order
Basic Concepts → SQL → ER Diagram → Joins → Normalization → Project → Advanced Topics


---

## 🛠️ Best Tools for Practice

| Tool | Purpose |
|------|---------|
| **MySQL Workbench** | Full database design & SQL editor |
| **XAMPP** | Local server with MySQL + phpMyAdmin |
| **phpMyAdmin** | Browser-based database management |
| **Visual Studio Code** | Write SQL with extensions |

---

## 🧠 Best Way to Learn
Theory (20%) + Daily SQL Practice (50%) + Small Projects (30%)

- Write SQL every day (even 15 minutes)
- Type every query yourself — no copy-paste
- Build one mini-project after each phase
- Learn from errors — that's where real growth happens
